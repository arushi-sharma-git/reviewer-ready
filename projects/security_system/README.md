#  Gatekeeper Security System
A modular Python system designed to validate security keys and log access attempts.

##  Features
- **Regex Validation:** Enforces strict 1-letter, 7-digit patterns.
- **JSON Persistence:** Stores history in a structured format.
- **Unit Tested:** Includes a full test suite for validation logic.
- **Modular Design:** Separation of concerns between logic and configuration.

##  Usage
1. Run the main system: `python json_vault.py`
2. Run security tests: `python test_suite.py`
